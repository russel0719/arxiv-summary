## 5. 개별 논문 요약

### 1. Memento: Reconstruct to Remember for Consistent Long Video Generation

- **arXiv**: https://arxiv.org/abs/2606.14667
- **Score**: 7.8 / 10
- **한줄 요약**: Memento는 장편 비디오 생성 시 발생하는 '아이덴티티 드리프트(identity drift)' 문제를 해결하기 위해, 피사체의 정체성 보존을 명시적인 '아이덴티티 그라운딩(identity grounding)' 작업으로 재정의한 프레임워크입니다.
- **핵심 기여**: 아이덴티티 보존과 시간적 연속성을 분리하여, 메모리 기반의 자동 회귀 생성 모델에서 흔히 발생하는 피사체 정보의 희석 문제를 해결하고 장기적인 일관성을 확보했습니다.
- **방법론 개요**: 피사체 재구성(subject reconstruction)을 위한 보조 학습 목표를 도입하고, 아이덴티티 정보와 문맥 정보를 분리하여 처리하는 '듀얼 쿼리 메모리 메커니즘'을 통해 장기 기억 내 피사체 특징을 명시적으로 유지합니다.
- **의의**: 기존의 컴퓨팅 비용이 높은 전역 어텐션 방식이나 단순한 압축 방식의 한계를 극복하여, 복잡한 서사 구조에서도 피사체의 일관성을 유지하며 확장 가능한 고품질 장편 비디오 생성을 가능하게 합니다.

### 2. MooMIns -- Monocular 3D Reconstruction and Object Pose Estimation from Multiple Instances

- **arXiv**: https://arxiv.org/abs/2606.14389
- **Score**: 7.8 / 10
- **한줄 요약**: 동일한 객체가 여러 번 등장하는 단일 이미지에서 객체의 3D 형상 복원과 6D 자세 추정을 동시에 수행하는 'MooMIns' 프레임워크 제안.
- **핵심 기여**: 학습된 사전 지식이나 CAD 모델 없이, 다중 인스턴스 간의 기하학적 일관성을 활용하여 객체의 3D 형상과 자세를 정밀하게 복원하는 최초의 프레임워크 구축.
- **방법론 개요**: 가우시안 스플래팅(Gaussian Splatting)을 역으로 활용하여, 여러 인스턴스를 단일 카메라 시점에서 렌더링하고 이를 통해 공통된 '표준 객체(Canonical Object)' 형상을 최적화하는 기하학적 복원 방식.
- **의의**: 기존의 데이터 기반 3D 복원에서 발생하는 '환각(hallucination)' 문제를 해결하고, 산업 현장의 빈 피킹(bin-picking) 등에서 고가의 센서 없이도 범용적이고 정밀한 3D 인식이 가능함.

### 3. CausalMotion: Structured Physical Reasoning as Keyframe and Trajectory Guidance for Training-Free Video Generation

- **arXiv**: https://arxiv.org/abs/2606.14317
- **Score**: 7.2 / 10
- **한줄 요약**: 비디오 생성 모델의 물리적 인과 관계와 장기적 일관성 문제를 해결하기 위해, VLM 기반의 추론과 물리적 궤적 계획을 결합한 학습 불필요(training-free) 프레임워크를 제안함.
- **핵심 기여**: 블랙박스 방식의 생성 패러다임을 '추론 후 생성'으로 전환하여, 명시적인 물리적 제약 조건을 통해 비디오의 시간적 일관성과 물리적 타당성을 획기적으로 개선함.
- **방법론 개요**: VLM을 통해 프롬프트를 인과적 키프레임과 객체 궤적으로 분해하고, 이를 사전 학습된 확산 모델의 노이즈 제거 과정에서 소프트 제약 조건으로 주입하여 물리적으로 타당한 영상을 생성함.
- **의의**: 기존 비디오 생성 모델의 고질적인 문제인 객체 순간 이동, 물리 법칙 위배, 장기적 일관성 결여를 모델 재학습 없이 효율적으로 해결하며, 생성 과정의 해석 가능성을 높임.

### 4. RepFusion: Leveraging Multimodal Priors for Denoising in Representation Space

- **arXiv**: https://arxiv.org/abs/2606.14700
- **Score**: 6.8 / 10
- **한줄 요약**: RepFusion은 픽셀 기반의 기존 확산 모델 대신, 사전 학습된 멀티모달 LLM(MLLM)과 표현 오토인코더(RAE)를 활용하여 노이즈가 섞인 시각적 표현을 직접 정제하는 새로운 생성 패러다임을 제안합니다.
- **핵심 기여**: 기존 VAE의 의미론적 한계를 극복하고, MLLM의 강력한 사전 지식을 활용하여 훨씬 적은 파라미터(1.3B)로도 기존 모델 대비 30% 향상된 생성 품질을 달성했습니다.
- **방법론 개요**: 학습된 RAE를 통해 시각적 표현을 추출하고, 이를 MLLM에 입력하여 노이즈가 포함된 표현을 정제합니다. 이 과정에서 생성된 멀티모달 임베딩을 Diffusion Transformer(DiT)의 AdaLN 모듈에 주입하여 이미지 생성을 수행합니다.
- **의의**: 단순한 텍스트 조건부 생성을 넘어, 멀티모달 이해와 생성 모델을 하나의 프레임워크로 통합함으로써 생성 모델의 효율성과 성능을 동시에 확보하는 새로운 설계 원칙을 정립했습니다.

### 5. One Layer's Trash is Another Layer's Treasure: Adaptive Layer-wise Visual Token Selection in LVLMs

- **arXiv**: https://arxiv.org/abs/2606.14277
- **Score**: 6.8 / 10
- **한줄 요약**: 기존의 고정적이고 비가역적인 시각 토큰 가지치기(pruning) 방식의 한계를 극복하기 위해, 계층별로 토큰의 중요도를 동적으로 재평가하고 선택적으로 처리하는 적응형 계층별 시각 토큰 선택(ALVTS) 프레임워크를 제안함.
- **핵심 기여**: 토큰을 영구적으로 삭제하지 않고 계층마다 재통합하는 방식을 통해 정보 손실 문제를 해결했으며, 모델 재학습 없이 89%의 토큰 압축률로 원본 성능의 96.7%를 유지하는 효율성을 입증함.
- **방법론 개요**: 각 계층 앞에 경량 토큰 선택 모듈을 배치하여 저랭크 근사(low-rank approximation)를 통해 토큰 중요도를 산출하고, 이를 바탕으로 활성 스트림과 스킵 스트림으로 분리하여 계산 효율성을 최적화함.
- **의의**: 대규모 시각-언어 모델(LVLM)의 추론 비용을 획기적으로 절감하면서도, 깊은 계층에서 필요한 세밀한 시각 정보를 보존함으로써 효율성과 정확도 사이의 최적 균형을 달성함.

### 6. BoRAD: Bootstrap your Own Representations for Multi-class Anomaly Detection

- **arXiv**: https://arxiv.org/abs/2606.14129
- **Score**: 6.8 / 10
- **한줄 요약**: 다양한 정상 범주를 하나의 모델로 학습하는 'One-for-all' 이상 탐지에서 발생하는 '동일성 지름길(Identity Shortcut)'과 '범주 간 혼동' 문제를 해결하기 위한 표현 능력 할당 프레임워크 제안.
- **핵심 기여**: 이상 탐지 실패 모드를 이론적으로 규명하고, 클래스 라벨이나 부정 샘플 없이도 학습 가능한 프로토타입 기반의 정규화 기법을 통해 고성능 다중 클래스 이상 탐지 모델을 구축함.
- **방법론 개요**: 학습 가능한 공유 프로토타입 뱅크를 도입하여 공간적 프로토타입 정렬과 상대적 글로벌 정렬을 수행하며, 추론 시에는 프로토타입을 제거한 경량화된 교사-학생 구조의 특징 불일치 방식을 사용함.
- **의의**: 기존의 복잡한 메모리 기반 방식이나 클래스별 개별 모델 운용 없이도, 단일 모델로 범주 간 간섭을 최소화하면서 이상치 탐지 성능을 극대화할 수 있는 효율적이고 확장 가능한 솔루션을 제공함.

### 7. ViT-Up: Faithful Feature Upsampling for Vision Transformers

- **arXiv**: https://arxiv.org/abs/2606.14024
- **Score**: 6.8 / 10
- **한줄 요약**: ViT-Up은 외부 인코더 없이 ViT 백본의 내부 계층적 표현을 활용하여 임의의 해상도로 특징을 업샘플링하는 암시적(implicit) 특징 예측 프레임워크입니다.
- **핵심 기여**: 기존 외부 가이드 방식의 특징 누출(feature leakage) 문제를 해결하고, 백본의 내부 정보를 활용하여 밀집 예측 성능을 극대화하며, 다양한 작업에 범용적으로 적용 가능한 확장성을 입증했습니다.
- **방법론 개요**: ViT의 중간 은닉 상태를 쿼리로 사용하여 좌표 기반의 암시적 디코더를 통해 고해상도 특징을 생성하며, 별도의 보조 인코더 없이 백본의 계층적 구조를 직접 활용합니다.
- **의의**: 파운데이션 모델의 효율성을 유지하면서도 고해상도 공간 추론 능력을 향상시켜, 복잡한 작업별 디코더 없이도 정밀한 밀집 예측(세그멘테이션 등)을 가능하게 합니다.

### 8. Conditioning Matters: Stabilizing Inversion and Attention in Diffusion Image Editing

- **arXiv**: https://arxiv.org/abs/2606.14125
- **Score**: 5.5 / 10
- **한줄 요약**: 확산 모델 기반 이미지 편집에서 텍스트 조건화(textual conditioning)가 확산 역학의 안정성과 편집 품질을 결정하는 핵심 요소임을 규명하고, 이를 최적화하는 SimEdit 프레임워크를 제안함.
- **핵심 기여**: 텍스트 조건화 정밀도가 확산 속도 필드(velocity field)의 안정성에 미치는 영향을 이론적으로 입증하고, 이를 통해 역전(inversion) 정확도와 편집 충실도를 동시에 개선하는 훈련 없는(training-free) 방법론을 제시함.
- **방법론 개요**: 조건화 신호를 정밀하게 다듬어 확산 궤적을 안정화하는 '조건화 정제'와, 편집 관련 토큰과 구조 보존 토큰을 분리하여 교차 분기 어텐션을 제어하는 '토큰 단위 어텐션 제어' 기법을 결합함.
- **의의**: 기존 편집 기법들이 겪던 역전 불안정성과 구조적 왜곡 문제를 해결함으로써, 복잡한 이미지 편집 작업에서도 원본의 배경을 유지하면서 정교한 의미론적 수정을 가능하게 하는 고성능 프레임워크를 제공함.
